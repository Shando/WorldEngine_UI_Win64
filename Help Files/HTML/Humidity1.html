Humidity


The Humidity tab has the following options:



Precipitation Weight        This is a number representing the Weighting to be applied for Precipitation. It must be between 0.00 and 99.99.

Irrigation Weight                This is a number representing the Weighting to be applied for Irrigation. It must be between 0.00 and 99.99.
